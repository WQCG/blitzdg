Detailed Description

template<typename P_expr1, typename P_expr2, _bz_typename P_numtype>
class div_et2< P_expr1, P_expr2, P_numtype >

Defines a stencil "div" ET that operates on two arrays of arbitrary type and specifies the return type as array<double, N_rank>.

The result type is used when running on an array and the etresult type when running on an expression. If you want to refer to the native type of the expression, set result="P_numtype" and etresult="typename T1::T_numtype". Sorry for that ugliness, but they define types differently.
