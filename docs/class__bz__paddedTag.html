Detailed Description

This tag class can be used to provide a nicer notation for constructing padded arrays: instead of Array<int,2> A(3, 3, GeneralArrayStorage<2>(paddedData)); one can simply write: Array<int,2> A(3, 3, paddedArray); where paddedArray is an object of type _bz_paddedTag.
