blitzdg
an open-source project aiming to implement parallel discontinuous Galerkin (dg) solvers for common partial differential equations systems using blitz++ for array and tensor manipulations and MPI for distributed parallelism.

25  /* A (P,Q) is in upper block triangular form. The kth block goes from
26  * row/col index R [k] to R [k+1]-1. The estimated number of nonzeros
27  * in the L factor of the kth block is Lnz [k].
28  */
29 
30  /* only computed if the AMD ordering is chosen: */

34  double *Lnz ; /* size n, but only Lnz [0..nblocks-1] is used */
35

37  int

39  nz, /* # entries in input matrix */
40  *P, /* size n */

42  *R, /* size n+1, but only R [0..nblocks] is used */
43  nzoff, /* nz in off-diagonal blocks */

52  * full structural rank */

521 /* klu_rgrowth : compute the reciprocal pivot growth */
522 /* -------------------------------------------------------------------------- */
523 
524 /* Pivot growth is computed after the input matrix is permuted, scaled, and
525  * off-diagonal entries pruned. This is because the LU factorization of each
526  * block takes as input the scaled diagonal blocks of the BTF form. The
527  * reciprocal pivot growth in column j of an LU factorization of a matrix C
528  * is the largest entry in C divided by the largest entry in U; then the overall
529  * reciprocal pivot growth is the smallest such value for all columns j. Note
530  * that the off-diagonal entries are not scaled, since they do not take part in
531  * the LU factorization of the diagonal blocks.
532  *
533  * In MATLAB notation:
534  *
535  * rgrowth = min (max (abs ((R \ A(p,q)) - F)) ./ max (abs (U))) */
536

538 (
539  int Ap [ ],
540  int Ai [ ],
541  double Ax [ ],
542  klu_symbolic *Symbolic,
543  klu_numeric *Numeric,
544  klu_common *Common /* Common->rgrowth = reciprocal pivot growth */
545 ) ;
