99 Purpose:
100 
101  Verifies and prints a row or column-oriented sparse matrix.
102 
103 Returns:
104 
105  UMFPACK_OK if Control [UMFPACK_PRL] <= 2 (the input is not checked).
106 
107  Otherwise (where n is n_col for the column form and n_row for row
108  and let ni be n_row for the column form and n_col for row):
109 
110  UMFPACK_OK if the matrix is valid.
111 
112  UMFPACK_ERROR_n_nonpositive if n_row <= 0 or n_col <= 0.
113  UMFPACK_ERROR_argument_missing if Ap and/or Ai are missing.
114  UMFPACK_ERROR_invalid_matrix if Ap [n] < 0, if Ap [0] is not zero,
115  if Ap [j+1] < Ap [j] for any j in the range 0 to n-1,
116  if any row index in Ai is not in the range 0 to ni-1, or
117  if the row indices in any column are not in
118  ascending order, or contain duplicates.
119  UMFPACK_ERROR_out_of_memory if out of memory.
120 
121 Arguments:
122 
123  Int n_row ; Input argument, not modified.
124  Int n_col ; Input argument, not modified.
125 
126  A is an n_row-by-n_row matrix. Restriction: n_row > 0 and n_col > 0.
127 
128  Int Ap [n+1] ; Input argument, not modified.
129 
130  n is n_row for a row-form matrix, and n_col for a column-form matrix.
131 
132  Ap is an integer array of size n+1. If col_form is true (nonzero),
133  then on input, it holds the "pointers" for the column form of the
134  sparse matrix A. The row indices of column j of the matrix A are held
135  in Ai [(Ap [j]) ... (Ap [j+1]-1)]. Otherwise, Ap holds the
136  row pointers, and the column indices of row j of the matrix are held
137  in Ai [(Ap [j]) ... (Ap [j+1]-1)].
138 
139  The first entry, Ap [0], must be zero, and Ap [j] <= Ap [j+1] must hold
140  for all j in the range 0 to n-1. The value nz = Ap [n] is thus the
141  total number of entries in the pattern of the matrix A.
142 
143  Int Ai [nz] ; Input argument, not modified, of size nz = Ap [n].
144 
145  If col_form is true (nonzero), then the nonzero pattern (row indices)
146  for column j is stored in Ai [(Ap [j]) ... (Ap [j+1]-1)]. Row indices
147  must be in the range 0 to n_row-1 (the matrix is 0-based).
148 
149  Otherwise, the nonzero pattern (column indices) for row j is stored in
150  Ai [(Ap [j]) ... (Ap [j+1]-1)]. Column indices must be in the range 0
151  to n_col-1 (the matrix is 0-based).
152 
153  double Ax [nz] ; Input argument, not modified, of size nz = Ap [n].
154  Size 2*nz for packed complex case.
155 
156  The numerical values of the sparse matrix A.
157 
158  If col_form is true (nonzero), then the nonzero pattern (row indices)
159  for column j is stored in Ai [(Ap [j]) ... (Ap [j+1]-1)], and the
160  corresponding (real) numerical values are stored in
161  Ax [(Ap [j]) ... (Ap [j+1]-1)]. The imaginary parts are stored in
162  Az [(Ap [j]) ... (Ap [j+1]-1)], for the complex versions
163  (see below if Az is NULL).
164 
165  Otherwise, the nonzero pattern (column indices) for row j
166  is stored in Ai [(Ap [j]) ... (Ap [j+1]-1)], and the corresponding
167  (real) numerical values are stored in Ax [(Ap [j]) ... (Ap [j+1]-1)].
168  The imaginary parts are stored in Az [(Ap [j]) ... (Ap [j+1]-1)],
169  for the complex versions (see below if Az is NULL).
170 
171  No numerical values are printed if Ax is NULL.
172 
173  double Az [nz] ; Input argument, not modified, for complex versions.
174 
175  The imaginary values of the sparse matrix A. See the description
176  of Ax, above.
177 
178  If Az is NULL, then both real
179  and imaginary parts are contained in Ax[0..2*nz-1], with Ax[2*k]
180  and Ax[2*k+1] being the real and imaginary part of the kth entry.
181 
182  Int col_form ; Input argument, not modified.
183 
184  The matrix is in row-oriented form if form is col_form is false (0).
185  Otherwise, the matrix is in column-oriented form.
186 
187  double Control [UMFPACK_CONTROL] ; Input argument, not modified.
188 
189  If a (double *) NULL pointer is passed, then the default control
190  settings are used. Otherwise, the settings are determined from the
191  Control array. See umfpack_*_defaults on how to fill the Control
192  array with the default settings. If Control contains NaN's, the
193  defaults are used. The following Control parameters are used:
194 
195  Control [UMFPACK_PRL]: printing level.
196 
197  2 or less: no output. returns silently without checking anything.
198  3: fully check input, and print a short summary of its status
199  4: as 3, but print first few entries of the input
200  5: as 3, but print all of the input
201  Default: 1
