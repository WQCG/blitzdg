10 /* CHOLMOD Partition module.
11  *
12  * Graph partitioning and graph-partition-based orderings. Includes an
13  * interface to CCOLAMD and CSYMAMD, constrained minimum degree ordering
14  * methods which order a matrix following constraints determined via nested
15  * dissection.
16  *
17  * These functions require METIS:
18  * cholmod_nested_dissection CHOLMOD nested dissection ordering
19  * cholmod_metis METIS nested dissection ordering (METIS_NodeND)
20  * cholmod_bisect graph partitioner (currently based on METIS)
21  * cholmod_metis_bisector direct interface to METIS_ComputeVertexSeparator
22  *
23  * Requires the Core and Cholesky modules, and three packages: METIS, CAMD,
24  * and CCOLAMD. Optionally used by the Cholesky module.
25  *
26  * Note that METIS does not have a version that uses SuiteSparse_long integers.
27  * If you try to use cholmod_nested_dissection, cholmod_metis, cholmod_bisect,
28  * or cholmod_metis_bisector on a matrix that is too large, an error code will
29  * be returned. METIS does have an "idxtype", which could be redefined as
30  * SuiteSparse_long, if you wish to edit METIS or use compile-time flags to
31  * redefine idxtype.
32  */
