Detailed Description

template<int N_rank>
class FortranArray< N_rank >

Class FortranArray specializes GeneralArrayStorage to provide Fortran style arrays (column major ordering, base of 1).

The noInitializeFlag() passed to the base constructor indicates that the subclass will take care of initializing the ordering_, ascendingFlag_ and base_ members.
