Detailed Description

This tag class can be used to provide a nicer notation for constructing unpadded arrays: instead of Array<int,2> A(3, 3, GeneralArrayStorage<2>(contiguousData)); one can simply write: Array<int,2> A(3, 3, contiguousArray); where contiguousArray is an object of type _bz_contiguousTag.
