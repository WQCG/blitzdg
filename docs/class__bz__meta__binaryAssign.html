Detailed Description

template<int I>
class _bz_meta_binaryAssign< I >

A metaprogram that uses the chunked_updater to assign an unknown-length expression to a pointer by unrolling in a binary fashion.

This way, we can get "almost-compile-time" unrolling of a length only known at runtime. I+1 is the number of significant bits in the longest length to consider. In this way, assigning a vector of length 7 is I=2 and will take 3 operations. The metaprogram counts down, that way it will start with large updates which will be aligned for aligned expressions.
