Detailed Description

Defines a set of utility functions for working with blitz arrays and their matrix and vector aliases.

Typedef Documentation

◆ ColumnMajorOrder

using blitzdg::ColumnMajorOrder = typedef blitz::ColumnMajorArray<2>

Used for constructing a matrix with column-major ordering.

The following example shows how to build a 5 x 5 real matrix with column-major ordering.

real_matrix_type mat(5, 5, ColumnMajorOrder());

◆ RowMajorOrder

using blitzdg::RowMajorOrder = typedef blitz::GeneralArrayStorage<2>

Used for constructing a matrix with row-major ordering.

The following example shows how to build a 5 x 5 real matrix with row-major ordering.

real_matrix_type mat(5, 5, RowMajorOrder());
Note
: Row-major ordering is the default ordering and does not need to be explicitly specified.
